YOURPARTS — Quick README
------------------------
Files:
- index.html         -> Static homepage (edit to set your WhatsApp number and email)
- products_sample.csv-> Sample product list for import
- README.txt         -> This file

How to publish on GitHub Pages:
1) In repo go to Settings → Pages
2) Under "Build and deployment" choose "Source: Branch = main, Folder = / (root)"
3) Save. Your site will appear at: https://raulavto44.github.io/yourparts/ within a minute or two.

Notes:
- Edit index.html and replace WH_NUMBER at top of script with your phone (no +).
- If you want product pages or search, later we can convert this to WordPress or add multiple static pages.
